# Employee Attrition Prediction

A machine learning project that predicts employee attrition using the IBM HR Analytics Employee Attrition dataset. This project aims to help organizations identify employees who are at risk of leaving by analyzing HR-related factors and building a predictive machine learning model.

---

## 📌 Overview

Employee attrition can lead to increased recruitment costs, loss of experienced employees, and reduced organizational productivity. This project applies machine learning techniques to analyze employee data, identify important factors influencing attrition, and predict whether an employee is likely to leave the company.

---

## ✨ Features

- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Feature selection
- Random Forest classification model
- Hyperparameter tuning using GridSearchCV
- Cross-validation for model evaluation
- Performance evaluation using multiple metrics
- Feature importance analysis

## 📈 Results

| Metric    | Score     |
| --------- | --------- |
| Accuracy  | **89%**   |
| AUC Score | **0.975** |

### Evaluation Metrics

- Accuracy
- Precision
- Recall
- F1-Score
- ROC-AUC

The optimized Random Forest model achieved strong overall performance while providing valuable insights into the key factors contributing to employee attrition.

---

## ⚠ Limitations

- The dataset is relatively small and may not represent all industries.
- The model showed lower recall for predicting employees who actually left due to class imbalance.
- Additional employee-related features could further improve prediction performance.
